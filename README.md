# 🎬 MovieInfo
 - 영화 검색 서비스

---
### 기능
- 트랜드 영화목록을 가져와서 CollectionView에 보여줍니다.
- 입력한 검색어에 대한 영화목록을 TableView로 보여줍니다.
- 각 셀을 클릭하면 해당 영화의 디테일 페이지를 보여줍니다.


---
### 사용 기술
 - RxSwift, SnapKit, ReactorKit, RxFlow, Moya
